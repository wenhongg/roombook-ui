# Room Booking UI

This is a UI for a room booking web application, built on [Material Kit React](https://demos.creative-tim.com/material-kit-react/#/?ref=mkr-readme). 

## Steps
To deploy the UI, run:
```
    npm install
    npm start
```
To generate production build, run:
```
    npm build
```
