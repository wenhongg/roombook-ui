# ROOMBOOK UI

This is a UI for a room booking web application, produced on [Material Kit React](https://demos.creative-tim.com/material-kit-react/#/?ref=mkr-readme). 

## STEPS TO DEPLOY

To deploy the UI, run:
```
    npm install
    npm start
```
To generate production build, run:
```
    npm build
```
